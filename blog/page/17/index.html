BitDefender Security Scan

May 17th, 2011

Компания Bitdefender выпустила BitDefender Security Scan - бесплатную утилиту для проверки Windows OS. Утилита позволяет проверить безопасность и стабильность системы, а также наличие ошибок реестра.

BitDefender Security Scan может работать в сочетании с любым продуктом безопасности, но он не может заменить вам антивирус или комплексный продукт, например Internet Security Suite. Он проверяет безопасность, проблемы реестра и необходимость обновления Windows с Microsoft Update. BitDefender Security Scan имеет размер 5.5 мб, так что вы легко можете взять его с собой на USB-flash. Для начала работы достаточно просто указать место установки сканера и нажать кнопку сканирования и компьютер будет проверен.

BitDefender Security Scan использует технологию BitDefender Quickscan. То есть при наличии подключения к интернет вам будет доступен анализ “облачными” технологиями для выявления вредоносных программ. При этом вся нагрузка ложиться на канал и сервера BitDefender, которые и хранят информацию о вирусах.

Основные возможности BitDefender Security Scan:

* “AV QuickScan” - сканирует компьютер на наличие вирусов, шпионских и других угроз

* Проверка реестра - сканирует компьютер на наличие недействительных записей реестра

* “Update watcher” - проверяет наличие обновлений для Windows

* Статус безопасности - проверяет текущее состояние безопасности вашей ОС

* Выборочная проверка на вирусы - позволяет мгновенно проверить любой подозрительный файл

* Запланированное сканирование - позволяет составить расписание проверок

Результат проверки моей системы на виртуальной машине:

Достоинство данного сканера в том, что он бесплатен и даст вам результаты своей проверки даром. Но вот для решения найденных проблем будет предложено установить полноценное программное решение (разумеется платное). В моём случае был предложен BitDefender Total Security 2011.
