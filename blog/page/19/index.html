Microsoft Safety Scanner 1.0.3001.0

Apr 16th, 2011

Вы думаете, что ваш компьютер заражен? Компания Microsoft выпустила инструмент для таких случаев - Microsoft Safety Scanner. Microsoft Safety Scanner - бесплатный инструмент безопасности, который вы можете загрузить с официальной страницы серверов Microsoft. Microsoft Safety Scanner обеспечивает сканирование по требованию и удаляет вирусы, шпионские программы и другое вредоносное ПО. Он работает с существующими антивирусным программным обеспечением вместе.

Примечание: Microsoft Safety Scanner имеет срок службы 10 дней после загрузки. Для повторного сканирования с последней защиты от вредоносного ПО, вам придется загрузить и запустить Microsoft Safety Scanner снова.

Важно: Microsoft Safety Scanner не является заменой антивирусного программного обеспечения, то есть для непрерывной защиты в реальном времени вам нужно выбрать другую программу. Например, Microsoft Security Essentials - может осуществлять защиту в режиме реального времени.

От себя хотел добавить, что интерфейс и режим работы нового продукта похож на Microsoft® Windows® Malicious Software Removal Tool, с парой отличий: обновление последнего происходит раз в месяц и, судя по размеру дистрибутива MSS 1.0, Microsoft Safety Scanner содержит большую базу вирусных сигнатур, скорее всего общую с Microsoft Security Essentials и Microsoft Forefront Endpoint Protection.

Firefox 6 Alpha 1 Russian

Apr 14th, 2011

13 апреля опубликована первая версия Firefox 6 alpha 1, разработка которого завершится 18 августа 2011.

Скачать его можно с Mozilla.org FTP: ftp.mozilla.org/pub/mozilla.org/firefox/nightly/latest-trunk/

Там же можно скачать пререлизную версию Firefox 5, которая пока идёт под номером 4.2 prealpha 1 и выйдет «на золото» 21 июня 2011 (детали возможного дизайна можно найти на сайте areweprettyyet.com). Firefox 7 выйдет также в 2011 году.

Вместе с тем же событием произошёл отказ от названия «ночных сборок» кодовым именем Minefield (англ., минное поле, что символизирует) — теперь они называются просто Nightly и, как и прежде, обновляются каждый вечер МСК.

Ссылки для прямой загрузки (все версии — на английском языке):

Скачать Firefox 6 alpha 1 для 32-битной Windows (13 Мб) Скачать Firefox 6 alpha 1 для OS X (34 Мб) Скачать Firefox 6 alpha 1 в tar/bzip2 для Linux i686 (14 Мб) Скачать Firefox 6 alpha 1 в tar/bzip2 для Linux x86_64 (15 Мб)

Mozilla разделит версии Firefox по четырём каналам, репозиториям Mercurial. Самый «свежачок» будет иметь место в mozilla-central (канал Nightly), далее нововведения, стабильность которых будет оценена как удовлетворительная — перейдут в firefox-aurora (канал Aurora), далее — в firefox-beta (канал Beta), и уже потом — в ветку релизных версий (канал Release).

Opera 11.10 Build 2092 Barracuda Final

Apr 12th, 2011

Opera 11.10 Build 2092 Barracuda Final - из блога http://my.opera.com

Итак, когда мы впервые представили вам Opera 11.10 “Barracuda”, вы увидели много новых функций, бесчисленные исправления и множество dev версий. Результатом всей этой работы в скором времени будет финальный релиз Barracuda. Парни сдержали слово -11 апреля 2011 года выпустили финальный релиз своего браузера.

Сегодня мы представляем“Barracuda” релиз-кандидат (далее RC). Этот RC содержит дополнительные исправления, чтобы “Barracuda” была еще более стабильной и отлаженной. Были исправлены несколько ошибок, приводящих к “падению” приложения, мы продолжаем отлаживать работу “Speed Dial”, исправляя в ней множество ошибок. Мы смогли снизить потребление Opera электроэнергии в Windows Vista и 7 для компьютеров работающих от батареи. То есть при тестировании производительности Opera, убедитесь, что вы подключили шнур питания к компьютеру. :)

В данный момент работа сосредоточена на решении важных проблем и ошибок Opera. Мы продолжаем работать и на будущие версии продукта, закладывая новые возможности и когда выйдет финальная версия “Barracuda” мы продолжим работу по исправлению других ошибок для будущих версий Opera. Сейчас мы сфокусированы на выпуске финальной версии для “Barracuda”. Спасибо вам за тестирование.

Локализации (переводы) обновлены в версии RC4.1.

Внимание: Перед тем как вы установите финальную версию Opera, удалите все предыдущие версии. У нас есть предположение, что часть экспериментального кода beta версий может повлиять на нормальную работу финальной версии.

Загрузить:

  • Opera 11.10 Build 2092 Barracuda Final для Windows
  • Opera 11.10 Build 2092 Barracuda Final для Mac
  • Opera 11.10 Build 2092 Barracuda Final для Linux/FreeBSD

Cервис по проверке файлов - проект “Валькирия”

Mar 25th, 2011

Comodo Group предоставляет новый бесплатный интернет-сервис по проверке файлов - проект “Валькирия”. “Валькирия” - это бесплатный Интернет-сервис проверки файлов ( http://valkyrie.comodo.com ), в облачной системе анализа файлов, который проверяет представленные пользователем файлы на наборе статических и динамических детекторов - в том числе эвристики, поиск файла и анализ поведения в режиме реального времени. С помощью “Валькирии”, наши пользователи смогут быстро получить заключение по безопасности неизвестного файла и могут помочь ускорить определение вредоносных программ “нулевого дня”.

Мы так гордимся, что наша“Валькирия” - услуга проверки файлов первой открывает публичные онлайн сервисы, которые обобщают различные виды методов ИИ ( AI Artificial intelligence -“Искусственный интеллект”), чтобы уведомить своих пользователей о том, что их представленный файл вредоносный или безопасный, с относительной вероятностью.

ВАЖНОЕ ДОПОЛНЕНИЕ: На данный момент: - сервис принимает PE-файлы (exe, dll, sys и т.д.) - Никаких проверок архивов пока нет.

  • Ограничение - 20 Mb
  • Внимательно смотрим на Final Result (на его принятии влияет и “Статический” анализ и “Динамический”).

Пакет обновления 1 (SP1) для Windows 7

Mar 16th, 2011

Пакет обновления 1 (SP1) для Windows 7 помогает сохранять максимальный уровень поддержки настольных компьютеров. Он также обеспечивает постоянное улучшение операционных систем (ОС) Windows, поскольку включает ранее выпущенные обновления, доставляемые из Центра обновления Windows, а также накопительные добавочные обновления платформ Windows 7, разработанные на основе отзывов пользователей и партнеров. Благодаря этому пакету организации могут развертывать единый набор обновлений.

Пакет обновления 1 (SP1) для Windows 7 предоставляет следующие преимущества.

• Обеспечение поддержки и обновления компьютеров. • Текущие обновления для платформы Windows 7. • Удобное одновременное развертывание накопительных обновлений. • Удовлетворение потребностей пользователей в большей мобильности бизнеса. • Полный набор инновационных средств виртуализации. • Более удобное развертывание пакета обновления, повышающее эффективность ИТ-структуры.
