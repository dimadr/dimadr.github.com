VBA32 Personal 3.12.14.1

Mar 8th, 2011

Приветствую всех участников dimadr.ru. Сегодня речь пойдет об антивирусном продукте ВирусБлокАда (VBA32) от белорусского разработчика ОДО “ВирусБлокАда”. Так сложилось, что данный продукт не очень популярен в нашей стране. Да и ориентирован он, в первую очередь, на белорусских потребителей. Но, учитывая, что в сети мне практически не встретился обзор данного продукта, я решил, все же, составить беглый обзор его возможностей.

Существует две версии программы для Windows Vista, Windows 7 и более ранних версий. Я буду тестировать программу на Windows 7, поэтому позволю себе кратко описать возможности именно этой версии продукта. В состав Vba32 Personal 3.12.14.1 для Windows Vista и Windows 7 входят:

* Файловый сканер. * Файловый монитор. * Система автоматических обновлений. * Расширение меню Проводника Windows. * Плагины для почтовых клиентов “The Bat!”, Microsoft Outlook, Microsoft Exchange Client. * POP3-фильтр для проверки принимаемых почтовых сообщений независимо от используемого почтового клиента. * Script-filter, защищающий от исполнения вредоносных скриптов в Microsoft Internet Explorer и Microsoft Outlook Express, а также любых других приложениях, использующих технологию Microsoft Windows Scripting Host.

Основные особенности антивируса для локальных персональных компьютеров:

* Проверка архивов, созданных наиболее употребляемыми архиваторами: RAR, ZIP, ARJ, TAR, GZIP, BZIP2, MS CAB. * Обработка (включая обезвреживание) сообщений в почтовых базах Microsoft Outlook Express 4 и 5, Microsoft Outlook, “The Bat!”. * Автоматическое обновление антивирусных баз и исполнимых модулей программы через Internet. * Автоматическое обновление до новых версий без необходимости переустановки комплекса. * Использование технологии “Deltapatching” для многократного снижения объема информации, скачиваемой из Интернета при обновлении. * Эвристический анализатор и технология распознавания вирусов “MalwareScopeТМ”, значительно повышающие эффективность борьбы с новыми вредоносными программами. * Применение технологии SOTeCheck, включающейся в работу при малой активности компьютера, для дальнейшего ускорения антивирусной обработки. * Эмулятор процессора с динамической трансляцией кода, эффективно обрабатывающий полиморфные, упакованные и зашифрованные вирусы, используется в универсальном распаковщике исполняемых файлов, обработанных различными программами защиты кода от исследования и программами-упаковщиками. * Контроль целостности и автоматическая подмена поврежденных модулей позволяет повысить надежность работы антивируса.

1. Установка Размер установочного файла 104 Мб. После запуска установщика можно выбрать один из 10 языков программы.

Далее предлагается выбрать один из способов установки – обычный, полный или выборочный. Я выберу выборочный, чтобы посмотреть какие компоненты можно установить.

Как видите, можно оперировать различными компонентами программы на свое усмотрение. Помимо этого, можно также установить дополнительные языки программы.

В общей сложности установка заняла около 3 минут. Далее появилось предложение о перезагрузке.

2. Первое знакомство После перезагрузки первое, что я увидел – предупреждение об активации. Замечу, что VBA32 – платный продукт для всех пользователей. Я буду работать в ознакомительном режиме.

После нажатия кнопки “Продолжить”, к моему удивлению, не последовало никаких реакций со стороны программы в виде всплывающих окон об обновлении антивирусных баз и подобной типичной информации. Пришлось вручную вызывать окно программы из трея.

Вот так, вполне аккуратно выглядит основное окно VBA32. Представляет оно из себя диспетчер задач, с помощью которого можно вызвать отдельный компонент программы. В какой то степени мне это напомнило работу продуктов Dr.Web, но в отличие от них, VBA32, имеет централизованное окно управления, каким и является данный диспетчер. Окно состоит из вкладок Общее, Инициализация и Дополнительно. Каждая вкладка содержит общие настройки программы.

На вкладке Общее находятся кнопки вызова отдельных компонентов программы, таких как Монитор, Сканер, Карантин, Планировщик. Также эти компоненты можно вызвать с помощью значка в системном трее Windows.

8. Выводы Плюсы: - Минимальные системные требования, практически незаметен для системы - Практически отсутствую всплывающие окна и сообщения различного рода (для начинающего пользователя, безусловно, плюс) - Имеет облегченный интерфейс, удобное управление.

Минусы: - Только базовые средства защиты - Нечастые обновления антивирусных баз - Негибкость настроек - Платный продукт

Антивирус VBA32 предоставляет базовую защиту компьютера от вирусов и интернет-угроз. Рассчитан на начинающего пользователя и обладает всеми необходимыми настройками соответствующего уровня. По моему мнению, имеет довольно легкий интерфейс, без лишних «красивостей». Не смотря на плюсы продукта, я не стал бы рекомендовать его для опытных пользователей в связи с отсутствием расширенных средств обеспечения безопасности и негибкости настроек.
