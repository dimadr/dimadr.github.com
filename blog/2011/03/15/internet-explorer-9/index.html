Финальная версия браузера Internet Explorer 9

Mar 15th, 2011

На мероприятии “Beauty of the web” 14 марта в городе Остин, США была анонсирована финальная версия нового браузера Internet Explorer 9. Основные изменения по сравнению с RC-версией не значительны. С тем как был анонсирован новый браузер, можно более подробно ознакомится на англоязычном сайте Internet Explorer 9.

Скачать новую версию бразуера и дополнительные компоненты можно по ссылкам:

* Windows Internet Explorer 9 for Windows 7

* Windows Internet Explorer 9 for Windows 7 64-bit Edition and Windows Server 2008 R2 64-bit Edition

* Windows Internet Explorer 9 for Windows Vista and Windows Server 2008

* Windows Internet Explorer 9 for Windows Vista 64-bit Edition and Windows Server 2008 64-bit Edition

* Windows Internet Explorer 9 Language Packs for Windows 7 and Windows Server 2008 R2

* Windows Internet Explorer 9 Language Packs for Windows Vista and Windows Server 2008

* Headers and Libraries for Windows Internet Explorer 9

Так же есть заметка в блоге президента Microsoft в России Николая Прянишникова о выходе данного браузера:

Во-первых, это быстрота и скорость. Сегодня за счёт использования новых технологий, которые раньше мы не применяли и много работали над ними, браузер стал очень быстрым. Вы не тратите время, он моментально загружается, и вы работаете быстрее, чем раньше.

Во-вторых, это новые системы безопасности. В Интернете повышается, увеличивается количество угроз – угроза слежения, угроза воровства личной информации, угроза списания ваших средства с онлайн счетов. Новый браузер от этого защищает – многоуровневая, многоступенчатая система защиты, обеспечения вашей безопасности. Поэтому использование нового браузера будет вас защищать.

Ну и наконец – функциональность: очень удобно. Как пример, я в своей работе использую функцию, когда вы можете перенести сайты, которые вы больше всего посещаете, как закладки в ваше меню Windows. И сегодня вы не тратите время на открытие браузера, не тратите время на поиск какой-то информации – он уже в вашем меню. Удобно, доступно и когда вы работаете с документами, вы можете быстро посмотреть любые обновления на тех сайтах, которые вам необходимы и это просто экономит вам время.
