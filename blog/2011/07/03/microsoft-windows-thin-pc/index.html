Microsoft Windows Thin PC Released

Jul 3rd, 2011

Корпорация Microsoft сообщила о том, что программная платформа Windows Thin PC (WinTPC) достигла стадии RTM (Release To Manufacturing); в ближайшее время начнётся её распространение. Ближайшее время - это 01 июля 2011 года. То есть распространение уже стартовало.

Загрузить пробную версию (триал 90 дней).

Краткая справка Новая операционная система по сути представляет собой «урезанный» вариант Windows 7, предназначенный для формирования тонких клиентов. ОС даст возможность корпоративным пользователям вдохнуть вторую жизнь в морально устаревшие компьютеры, которых уже не хватает для удовлетворения потребностей пользователей.

WinTPC позволяет применять специальные фильтры записи, запрещающие локальную инсталляцию приложений и хранение данных. Кроме того, ИТ-администраторы смогут запретить использование определённых комбинаций клавиш — например, сочетания Ctrl + Alt + Delete, при помощи которого можно вызвать «Диспетчер задач Windows».

Представляет из себя операционную систему, предназначенную для создания тонких клиентов. Грубо говоря, Windows Thin PC позволит конечным пользователям (а ими будут предприятия с подпиской Software Assurance) использовать старые компьютеры как нересурсоемкие терминалы для подключения к высокопроизводительным серверам.

Тем не менее, Windows Thin PC привносит рад новшеств на платформу Windows Embedded, самым значимым из которых, пожалуй, является технология RemoteFX и поддержка System Center Configuration Manager. RemoteFX позволит использовать мультимедийные возможности семейства NT 6.1 в среде терминального доступа Windows Thin PC, а поддержка System Center Configuration Manager позволит легко централизованно управлять развертыванием системы. Ещё одной примечательной возможностью Windows Thin PC является наличие фильтра файловой системы, предотвращающего запись данных на жесткий диск пользователем или приложениями, что положительным образом может сказаться на безопасности конечных решений.

Проще говоря, семерка с SP1 на базе Embedded для слабых машин, размер wim файла ~800 мб. В ней изначально отсутствует Медиа центр, макер, Планшетники и связанные с ними речь, текст, ножницы. Нет даже в меню пуск ярлыков калькулятора, блокнота, WordPad, Paint. Нет оценки и восстановления системы, включения-отключения компонентов. Есть только IE 8.0 и WMP, а также Аеро со стандартными темами и муз. темы. Выпущена лишь на пяти языках. Только 32 бит редакция, ибо 64 бит предполагает использование более мощных машин. Допускает установку только из под DOS. Поначалу операционная система для тонких клиентов будет доступна только на английском языке; планируется ли выпуск локализованных версий, пока не сообщается.
