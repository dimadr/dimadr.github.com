Panda 2012 Security Products - 90 дней бесплатно [обновлено]

Jun 11th, 2011

Компания Panda Security в содружестве с компанией Майкрософт предоставляют возможность протестировать антивирусный продукт Panda Internet Security 2012 в течении 90 дней бесплатно. Для этого вам необходимо скачать специальную версию - Download installer: MicrosoftpromoIS12.exe (Free 90 days OEM).

Установите программу и получите бесплатную защиту Panda Internet Security 2012 на 3 месяца. Вводить серийный лицензионный ключ или код активации не требуется. Комплексный антивирус Panda IS 2012 совместим с ОС Windows 7 (32/64-bit), Vista (32/64-bit) и Windows XP (32-bit).

проверено, работает…

Краткая справка Panda Internet Security 2012 сочетает в единой программе антивирус, антишпион, фаервол, защиту персональных данных, антиспам, родительский контроль, резервное копирование, режим Игры/Мультимедиа, управление домашней сетью, удаленный доступ к системе и веб-защиту. Panda IS 2012 обеспечивает защиту от всех вредоносных программ и онлайн-угроз.

Основные возможности и улучшения Panda Internet Security 2012: • Улучшенная защита от всех Интернет-угроз • Улучшенная технология Коллективного разума • Улучшенная технология поведенческого анализа • Улучшенный персональный фаервол • Улучшенный инструмент Panda USB Vaccine • Добавлена песочница для веб-браузера Panda Safe Browser (Sandboxing) • Улучшенная защита от спама • Улучшенный Родительский контроль • Улучшенный антивирусный загрузочный диск Panda SafeCD • Виртуальная клавиатура • Режим Игры/Мультимедиа • Управление домашней сетью • Защита персональных данных • Резервное копирование и восстановление • 2GB для онлайн резервного копирования • Удаленный доступ к компьютеру • Обновления в режиме реального времени
