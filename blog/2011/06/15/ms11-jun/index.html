Microsoft Security Bulletin для Windows XP SP3 от 14 июня 2011 г.

Jun 15th, 2011

Microsoft Security Bulletin Advance Notification от 14 июня 2011 г. Предварительно - обновлений для Windows XP SP3 будет 10 (одно из них MRT).

KB890830 - v3.20 - Microsoft® Windows® Malicious Software Removal Tool 14 июня 2011 года KB2476490 - Уязвимость в OLE Automation делает возможным удаленное выполнение кода KB2478656 - Уязвимость в .NET Framework and Microsoft Silverlight делает возможным удаленное выполнение кода KB2478658 - Уязвимость в системе безопасности платформ .NET Framework версий 2.0 с пакетом обновления 2 (SP2) и 3.5 с пакетом обновления 1 (SP1) в Windows XP x86 делает возможным удаленное выполнение кода KB2478663 - Уязвимость в системе безопасности .NET Framework 4 в Windows XP делает возможным удаленное выполнение кода KB2503665 - Уязвимость в Ancillary Function Driver делает возможным несанкционированное получение прав KB2530548 - Накопительное обновление системы безопасности для Internet Explorer 6, 7, 8 в Windows XP KB2535512 - Уязвимости в Distributed File System делают возможным удаленное выполнение кода KB2536276 - Уязвимости в клиенте SMB делают возможным удаленное выполнение программного кода KB2538242 - MS11-025:Уязвимость в Microsoft Foundation Class (MFC) Library for C++ 2005 SP1 делает возможным удаленное выполнение кода: June 14, 2011 KB2538243 - MS11-025: Уязвимость в Microsoft Foundation Class (MFC) Library for Visual C++ 2008 SP1 делает возможным удаленное выполнение кода: June 14, 2011 KB2544521 - Уязвимость в Internet Explorer 6, 7, 8 компонент Vector Markup Language делает возможным удаленное выполнение кода KB2544893 - Уязвимость в Internet Explorer 6, 7, 8 компонент - MHTML может привести к распространению информации ISO-образ, содержащий обновления системы безопасности за июнь 2011 г.

p.s. В ISO собраны обновления за июнь 2011 года для Windows XP SP3 x86 русской и английской, а также для Windows 7 x86 и x64. Содержимое хорошо видно на картинке.Удалены обновления для других ОС, добавлены фиксы Net.Framework’ов и Visual C++ 2005, Visual C++ 2008.
