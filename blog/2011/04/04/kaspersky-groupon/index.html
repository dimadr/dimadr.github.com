Kaspersky Internet Security 2011 11.0.2.556 - 3 месяца бесплатно

Apr 4th, 2011

Компания ЗАО «Лаборатория Касперского» совместно с Groupon, Inc. проводят акцию. Подписчики Groupon имеют возможность совершенно бесплатно получить промо-код, который можно обменять на Kaspersky Internet Security 2011 для защиты двух компьютеров на 3 месяца.

Для участия в акции вы должны быть зарегистрированы на сайте groupon.ru/signup. Получить купон вы сможете на следующей странице. Получив купон, перейдите на страницу http://promo.kaspersky.com/grupon заполните и дождитесь письма с кодом активации KIS 2011. Условия акции подробно изложены на этой странице.

Внимание: Обменять промо-код купона на код активации для Kaspersky Internet Security 2011 необходимо до 3 мая 2011 г. (включительно). Активировать полученный код активации для Kaspersky Internet Security 2011 необходимо до 31 августа 2011 г. (включительно). Дистрибутив для акции используется специально подготовленный, загрузить его можно с сайта http://www.kaspersky.ru/groupon.

проверено, работает…
