Microsoft Security Bulletin для Windows XP SP3 от 12 апреля 2011 г.

Apr 13th, 2011

Microsoft Security Bulletin Advance Notification от 12 апреля 2011 г. Предварительно - обновлений для Windows XP SP3 будет 14 (четырнадцать!), одно из них MRT. 7 обновлений в статусе - “Критическое”; 7 обновлений в статусе - “Важное”.

KB890830–v3.18 – Microsoft® Windows® Malicious Software Removal Tool Windows XP: April 11, 2011 KB2412687-MS11-029: Уязвимости в системе безопасности GDI+ во всех выпусках Windows XP. KB2485663-MS11-033: Уязвимость в текстовых конвертерах Wordpad делает возможным удаленное выполнение кода. KB2491683-MS11-024: Обновление безопасности для редактор титульных страниц факсов. KB2497640-MS11-018: Накопительное обновление для системы безопасности браузера Internet Explorer 6. KB2497640-MS11-018: Накопительное обновление для системы безопасности браузера Internet Explorer 7. KB2497640-MS11-018: Накопительное обновление для системы безопасности браузера Internet Explorer 8. KB2503658-MS11-026: Уязвимость в MHTML может привести к распространению информации. KB2506212-MS11-024: Уязвимость в Windows Fax Cover Page Editor MFC components делает возможным удаленное выполнение кода. KB2506223-MS11-034: Уязвимости в драйверах режима ядра Windows делают возможным несанкционированное получение прав. KB2507618-MS11-032: Уязвимость в драйвере OpenType CFF делает возможным удаленное выполнение кода. KB2508272-MS11-027: Накопительное обновление, устанавливающее биты аннулирования для элементов ActiveX. KB2508429-MS11-020: Уязвимости в сервере SMB могут сделать возможным удаленное выполнение кода. KB2509553-MS11-030: Уязвимость в DNS Resolution может привести к удаленному выполнению кода. KB2510531-MS11-031: Уязвимости в JScript и VBScript v5.8 scripting engines могут привести к удаленному выполнению кода. KB2510581-MS11-031: Уязвимости в JScript и VBScript v5.8 scripting engines могут привести к удаленному выполнению кода. KB2511455-MS11-019: Уязвимости в клиенте SMB делают возможным удаленное выполнение программного кода.

Загрузить обновления для Windows XP SP3 x86: April 12, 2011 - одним архивом.
