Microsoft Safety Scanner 1.0.3001.0

Apr 16th, 2011

Вы думаете, что ваш компьютер заражен? Компания Microsoft выпустила инструмент для таких случаев - Microsoft Safety Scanner. Microsoft Safety Scanner - бесплатный инструмент безопасности, который вы можете загрузить с официальной страницы серверов Microsoft. Microsoft Safety Scanner обеспечивает сканирование по требованию и удаляет вирусы, шпионские программы и другое вредоносное ПО. Он работает с существующими антивирусным программным обеспечением вместе.

Примечание: Microsoft Safety Scanner имеет срок службы 10 дней после загрузки. Для повторного сканирования с последней защиты от вредоносного ПО, вам придется загрузить и запустить Microsoft Safety Scanner снова.

Важно: Microsoft Safety Scanner не является заменой антивирусного программного обеспечения, то есть для непрерывной защиты в реальном времени вам нужно выбрать другую программу. Например, Microsoft Security Essentials - может осуществлять защиту в режиме реального времени.

От себя хотел добавить, что интерфейс и режим работы нового продукта похож на Microsoft® Windows® Malicious Software Removal Tool, с парой отличий: обновление последнего происходит раз в месяц и, судя по размеру дистрибутива MSS 1.0, Microsoft Safety Scanner содержит большую базу вирусных сигнатур, скорее всего общую с Microsoft Security Essentials и Microsoft Forefront Endpoint Protection.
