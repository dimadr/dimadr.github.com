Opera 11.52 "Swordfish" Build 1100 Final

Oct 19th, 2011

19 октября 2011 года - норвежские разработчики представили новую версию “Swordfish” Opera - Opera 11.52 build 1100 “Swordfish” Final.

Работа над Opera 12.00 (Wahoo) продолжается, но поступающие данные говорят о наличии ошибок в Opera 11.51 build 1087 Final. Поэтому мы не бросаем тот релиз и продолжаем его “полировать”. Сегодняшний пакет Opera 11.52 “Swordfish” build 1100 Final, своего рода надстройка над финалом и должна придать ему стабильность и надежность. То есть сегодняшний build ставиться на Stable версию поверх, а не на “Wahoo” Opera Next. Это рекомендуемое обновление предыдущей сборки “Swordfish”. Полный список изменений вошедший в build 1100 можно прочесть здесь.

Лог изменений в 11.52 build 1100: Display and scripting • Adjusting volume on a YouTube HTML5 Video causes freeze • Fixed a non-exploitable bug which allowed injection of untrusted markup into the X-Frame-Options error page, as reported by Masato Kinugawa. Network • Crashes when downloading via BitTorrent Security • Fixed an issue where manipulating fonts in SVG could allow execution of arbitrary code
