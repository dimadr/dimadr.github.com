Panda Cloud Antivirus 1.9.2 Beta

Jun 4th, 2012

04 июня 2012 - Panda Cloud Antivirus 1.9.2 Beta. (Подронее под катом) 24 апреля 2012 - узнал приятное подтверждение тому, что разработка Panda Cloud Antivirus идет “полным ходом”.(подробнее под катом)

Изменения в Panda Cloud Antivirus 1.9.2 Beta: • Новый фаервол для Windows 7 и Windows 8. • Новая (IDS) система обнаружения вторжений для защиты от сетевых атак. • Поведение фаервола может адаптироваться по гео зонам. • Улучшена регистрация фаервола в “Центре безопасности Windows”. • Фаервол Panda Cloud Antivirus может использоваться опционально - “включено” или “выключено”. • Новые технологии обнаружения и удаления вредоносных программ. • Улучшена технология обнаружения вредоносных программ в ОЗУ. • Улучшено обнаружение и удаление упакованных вредоносных программ. • Улучшено управление карантином. • Мониторинг за процессами опционально может использоваться без URL-мониторинга. • Монитор активности антивирусной защиты отображается в главном окне программы. • Обнаруженные опасности отображаются в главном окне программы. • Добавлено несколько дополнительных параметров в настройках работы антивирусного модуля. • Улучшен алгоритм сканирования и взаимодействие с карантином, а также модулем сканирования ОЗУ. • Интегрирован поиск Google (Yahoo удален). • Улучшена регистрация антивируса в Центре Безопасности Windows. • Улучшено взаимодействие с системой “Коллективного Разума” (сканирование “в облаке”). • Снижена нагрузка на систему при лечении файлов.

Загрузить Panda Cloud Antivirus 1.9.2 Beta.

На сайте разработчика появилась страница голосования - выбор нового интерфейса для Panda Cloud Antivirus.

30 апреля 2011 года особый день в Panda Security по двум причинам. Первый, потому что сегодня исполняется двухлетний юбилей с момента выпуска первой бета-версии Panda Cloud Antivirus, первого бесплатного антивируса с облачными технологиями. В честь этого, Panda Security выпускает другую бета-версию сегодня, Panda Cloud Antivirus Beta 1.9.1.

Цель этой бета-версии состоит в том, чтобы протестировать перехват сетевого трафика, чтобы создать основанную на облаке функциональность брандмауэра в будущих версиях. В этой бете сетевой перехват применен и к HTTP и к трафику POP3. Даже при том, что программа включает Монитор Процесса, который отслеживает запросы веб-трафика HTTP на основе для каждого процесса и учитывает блокирование для каждого процесса, никакая традиционная функциональность брандмауэра не обеспечена в бете, только перехват.

Чтобы способствовать более активному тестированию этой версии, Panda Security выдает 10 Amazon.com подарочных сертификатов номиналом 100$ самым активным испытателям бета-версии, тем, кто будет сообщать об ошибках в форум поддержки Panda Cloud Antivirus Beta. Победители подарочных сертификатов будут выбраны в конце тестирования модераторами форума поддержки Panda Cloud Antivirus Beta среди пользователей с большим количеством новых (неизвестных во время регистрации) отчетах об ошибках. Отметьте, что бета-тестирование может продолжиться после окончания промо-акции (1-ого августа 2011). Кроме того, активным испытателям бета-версии будут вручены подарочные коды лицензии PRO версии нового продукта.
