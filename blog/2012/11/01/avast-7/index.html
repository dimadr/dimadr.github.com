Avast! 7.0.1474 Final

Nov 1st, 2012

31 октября 2012 - avast! 7.0.1474 Final.

<code>
<a href="http://forum.avast.com/index.php?topic=108170.0" target="_blank">Изменения в версии 7.0.1474:</a>
31.10.2012 (7.0.1474)
• Устранена причина "синих экранов в Windows 8 (остались еще ошибки, они на изучении).
• Исправлены падения и зависания продукта в Windows XP 64b.
• Исправлены ошибки в модуле "WebRep".
• Улучшена стабильность модуля "Песочница".

23.10.2012 (7.0.1473)
• Закончена отладка элементоа METRO UI (уведомления, иконки и т.д.)
• Плагин WebRep теперь работает корректно во всех основных браузерах.
• Исправлены проблемы сети в Windows 8.
• Исправлена разные падения/BSODs основанные на логах BETA версии продукта.
</code>

Новые возможности версии Avast 7.0: • В Windows 7, теперь вы можете переключаться между горизонтальной и вертикальной ориентацией гаджета рабочего стола. • Avast! Аккаунт портала (https://my.avast.com) будет завершен в течение следующих нескольких дней, базовая функциональность уже есть. Наслаждайтесь. • Autosandbox malware анализ еще не полностью функционален (он будет включен (через VPS) в ближайшее время) • Новый установщик • Улучшения в интерфейсе • Сервис FileRep (на основе облака) • Потоковое обновления • Улучшения песочницы и автоматической песочницы (autosandbox) • Улучшение защиты браузера • Удаленная помощь для помощи вашим друзьям с проблемами на компьютере • Инструмент поддержки (собирает лог-файлы и т.д. вместе, что позволяет представить их avast! для устранения неполадок) • Экспорт / Импорт настроек • Улучшение заставки • Работает на Win8 Developer Preview • И многие другие улучшения безопасности • Исправлен удалённый доступ avast! • Внесены исправления в файлы помощи и локализаций • Исправлена регистрация продукта • Заработала страница настроек аккаунта

Большим плюсом данного продукта считаю наличие в бесплатной версии avast! 7.0 Free Edition облачных технологий. Можно назвать еще пару вендоров с бесплатным “облаком”, но с таким функционалом пока нет на рынке продукта.
