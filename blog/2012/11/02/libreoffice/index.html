LibreOffice.org 3.6.3 Final Multilingual

Nov 2nd, 2012

26 октября 2012 года - LibreOffice.org 3.6.3 Final Multilingual (русская справка).

Краткая справка LibreOffice — это качественный и полностью бесплатный офисный пакет со всеми необходимыми компонентами. Построен на основе популярного офисного пакета OpenOffice и развивается сообществом свободных разработчиков. Основной целью проекта является создание полностью свободного и независимого офисного пакета с большим сообществом разработчиков. Если OpenOffice развивается под контролем компании Oracle, то LibreOffice избавлен от жестких рамок и полностью свободен (в рамках лицензии LGPL).

Программа способна заменить платный офисный пакет Microsoft Office. LibreOffice включает все, что нужно пользователю: текстовый процессор, электронные таблицы, программа для создания презентаций, векторный графический редактор, редактор математических формул и механизм подключения к внешним базам данных. Также в пакете имеется ряд фильтров для импорта и экспорта документов в наиболее распространенные форматы, включая форматы MS Office. Все компоненты пакета включают все необходимое для удобной и продуктивной работы. Подробный перечень функций и изменений новой версии.

Дистрибутив LibreOffice for Windows, переупакованный в административную точку. Собран в тихую установку с автоинсталляцией. Внимание. Для установки приложения потребуется установленный .NET Framework 4.0 для Windows XP SP3 или .NET Framework 4.5 для Windows 7 SP1. Для нормальной работы приложения потребуется Java™ SE Runtime Environment Sun Java™ Runtime 6 Update 37 или Sun Java™ Runtime 7 Update 9.

Дистрибутив LibreOffice 3.6.3 Stable Multilingual for Windows, переупакованный в административную точку с русской справкой. Собран в “тихую” установку с автоинсталляцией: Загрузить LibreOffice.org 3.6.3 Stable Multilingual (интегрирована русская справка); зеркало 2 Ключи тихой установки: -ai -fm0 Размер=156.37 Mb MD5=6ad625d79446c161c476ad1610154197 SHA1=ea7814cda8e2ea2b0a71251cb031269a71a7377f

Дистрибутив Java™ SE Runtime Environment 6 Update 37 - 7 Update 9 x86 / x64. Может использоваться в качестве тихой установки, аддона и т.п., как при инсталляции ОС, так и на “живую” систему. Во время установки удаляет предыдущую версию Java (так настоятельно рекомендует Oracle).
