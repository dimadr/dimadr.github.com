Mac OS X 10.8.1 Mountain Lion

Aug 23rd, 2012

23 августа 2012 - немного раньше, чем все ожидали появился релиз Mac OS X 10.8.1 Mountain Lion (12B19). Вы можете загрузить обновление в App Store и там же найти новый дистрибутив.

<code>
Основные изменения:
• Resolve an issue that may cause Migration Assistant to unexpectedly quit
• Improve compatibility when connecting to a Microsoft Exchange server in Mail
• Address an issue playing audio through a Thunderbolt display
• Resolve an issue that could prevent iMessages from being sent
• Address an issue that could cause the system to become unresponsive when using Pinyin input
• Resolve an issue when connecting to SMB servers with long names
• Address a issue that may prevent Safari from launching when using a Proxy Automatic Configuration (PAC) file
• Improve 802.1X authentication with Active Directory credentials.</code>

В данном списке отсутствует информация об изменениях в автономной работе. Но уже есть сравнительная таблица, наглядно характеризующая произошедшие улучшения.
