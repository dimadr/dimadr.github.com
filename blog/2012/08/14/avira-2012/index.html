Новые продукты Avira доступны для загрузки сегодня - 15 февраля 2012. Основные продукты Avira AntiVir Personal, Avira AntiVir Premium, Avira AntiVir Premium Security Suite, Avira AntiVir Professional поменяли свои названия на Avira Free Antivirus, Avira Antivirus Premium 2012, Avira Internet Security 2012 и Avira Professional Security соответственно. Также изменил название и Avira AntiVir Server, в 2012 версии он вышел как Avira Server Security и поддерживает 4 основных мировых языка.

Изменение в новой версии 2012: Удобство использования - Переработанный графический интерфейс, отличающийся большей чистотой линий и ориентированный на интуитивное понимание - Полностью настраиваемый режим презентации для бизнес-пользователей - Дополнительные функции для домашних пользователей в игровом режиме - Упрощенная экспресс-установка продуктов

Защита данных - Более функциональная система обнаружения угроз - Улучшенная функция лечения зараженных файлов - Интерактивная поддержка через TeamViewer - Более сильные параметры защиты по умолчанию - Автоматическое продление лицензии - Полная поддержка IPv6 - Более ненавязчивая система сообщений (настраиваемая во всех платных продуктах)

Стабильность - Более быстрые системы инициализации и сканирования - Возможность сканирования в режиме пониженного приоритета - Возможность автоматического удаления при установке несовместимого программного обеспечения безопасности.

Avira ProActiv is not available for 64 Bit systems at the moment

The new behaviour-based detection technology Avira ProActiv, which is integrated into the the products Avira Antivirus Premium 2012, Avira Internet Security 2012 and Avira Professional Security right now is only available for 32 bit systems. There are technical reasons which prevent the support on 64 bit systems:

The module right now uses so called kernel hooks to monitor the system. Those kernel hooks are not available on 64 bit systems. Work on a technical solution for Windows Vista and Windows 7 64 bit systems are in progress. However, at the moment no statement concerning the availability can be given.
