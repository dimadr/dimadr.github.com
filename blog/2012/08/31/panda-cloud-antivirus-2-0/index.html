Panda Cloud Antivirus 2.0.1

Aug 31st, 2012

Вышла новая версия Panda Cloud Antivirus 2.0.1; зеркало 2 - “пионер” облачной защиты.

Признаюсь, я голосовал на сайте за пункт ”А”, а дизайн судя по тестовой установке немного другой. Тоже симпатичный и информативный, как мне кажется. Файрвол работает только в версии Pro, как и USB “Вакцина”.

<code>
Изменения в версии 2.0.1:
• Шрифт интерфейса увеличен на 125%.
• Исправлена ошибка определения адреса в описаниях (модуль PSUN отчетов).
• Исправлена ошибка отображения подсказки в отчетах брандмауера.
• Исправлена ошибка замедления доступа к расшаренным ресурсам.
• Исправлена ошибка замедления отправки почты из Thunderbird.
• Исправлена ошибка замедления доступа к файлам в сети.

• Interface fonts at 125%. Correction in the Pixels per Point (PPP) escalation process.
• Wrong detection URL in tooltips (PSUN reports).
• Error when displaying tooltips regarding firewall reports.
• Slowdown when accessing shared resources. All documents where being scanned on the server each time the shared folder was being accessed.
• Slowdown when sending mails from Thunderbird.
• Slowdown when accessing files on the network.

<a href="http://blog.cloudantivirus.com/antivirus/release/" target="_blank">Новое версии 2.0</a>:
• Новый интерфейс Panda Cloud Antivirus 2.0.
• Добавлен файрвол (про версия).
• Улучшена производительность продукта оптимизацией кода.
• Сканирует на 50% быстрее. 
• Новый антивирусный движок на основе облачной технологии. 
• Новый поведенческий движок, содержит много инноваций.
• Новые дополнительные параметры настройки, например, подтверждение перед нейтрализацией вредоносного объекта.
• Улучшена защита при отсуствии интернет-соединения.
• Process Monitor позволяет контролировать интернет-соединение для каждого процесса.
• Продукт совместим с Windows 8.
• Добавлено управление лицензией.

The main new features which Panda Cloud Antivirus 2.0 incorporates over the existing 1.5.2 version are the following:
• Community-based firewall (pro edition only).
• New cloud-based disinfection engine.
• New behavioral analysis engine in the free edition.
• Improved offline protection.
• Process monitor allows seeing Internet connections on a per process basis.
• Improved performance and 50% faster scans.
• New advanced configuration options such as asking for confirmation before neutralizing.
• Compatibility with Windows 8.
• New interface.
• Ability to manage different licenses.</code>
