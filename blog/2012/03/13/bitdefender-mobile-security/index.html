Bitdefender Mobile Security для Android

Mar 13th, 2012

Сейчас все вендоры спешат занят свою нишу на рынке антивирусных решений для мобильных устройств - компания Bitdefender не исключение. Один из лидеров антивирусного программного обеспечения выпускает две версии антивируса: бесплатную (free) и Premium (only $9.95/year).

Сравнительную характеристику версий смотрите ниже на фото:

То есть в Premium версии добавлены функции “Интернет безопасности” и “Антивор”, при этом “Удаленная геолокация” есть и в бесплатной версии.
