Microsoft Security Essentials 4.0.1512.0 Beta

Mar 12th, 2012

29 ноября 2011 года стартовало публичное тестирование Microsoft® Security Essentials. Если вы оставляли заявку на участие в бета - программе и вас выбрали, то вы сможете загрузить новый продукт и помочь в его разработке.

What’s New in Microsoft Security Essentials 4.0.1111.0 Beta : • Enhanced protection through automatic malware remediation: The Beta program will clean highly impacting malware infections automatically, with no required user interaction. • Enhanced performance: The Beta includes many performance improvements to make sure your PC performance isn’t compromised. • Simplified UI – Simplified UI makes Microsoft Security Essentials Beta easier to use. • New and improved protection engine: The updated engine offers enhanced detection with cleanup capabilities and better performance.

Заявлены следующие изменения: • Улучшенная защита с помощью автоматического исправления вредоносных программ: бета-версия программы будет удалять вредоносные инфекции автоматически, без
взаимодействия с пользователем. • Повышенная производительность: Beta включает в себя множество улучшений производительности. • Упрощенный пользовательский интерфейс - упрощенный пользовательский интерфейс делает Microsoft Security Essentials Beta проще в использовании. • Новый усовершенствованный движок: обновленный движок предлагает усовершенствованные функции обнаружения с возможностями очистки и лучшую производительность.
