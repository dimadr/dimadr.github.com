CCleaner 3.24 Build 1850 Final

Oct 25th, 2012

25 октября 2012 года - обновление версии Piriform CCleaner 3.24 build 1850.

Изменения в версии CCleaner:

<code>
v3.24.1850 (25 Oct 2012)
• Добавлено управление плагинами и расширениями Firefox.
• Оптимизирована архитектура программы для версии 64-bit для Windows 7 и 8.
• Улучшена очистка cookie для Firefox Aurora.
• Улучшена очистка "сохраненных форм ввода" для Chrome Canary.
• Добавлена очистка для AVG AntiVirus 2013, 4Sync, Copernic, DVDFab, Inkscape и PKZip.
• Улучшена очистка для Windows MRUs.
• Автоматическое обновление для версии Pro происходит раньше.
• Небольшие (незначительные) улучшения интерфейса программы.
• Исправлена незначительная ошибка.
</code>

Ссылка на загрузку разных версий с сайта производителя: http://www.piriform.com/ccleaner/builds, позднее там появиться “slim” и “Portable” версии программы.

Краткая справка CCleaner - утилита для очистки системы от мусора: cookies, история посещения сайтов в IE, временные файлы Интернета, строки поиска, файлы Корзины, записи неиспользуемых расширений в системном реестре, библиотеки, уже удаленных программ и др. Несмотря на отсутствие «наворотов», свойственных другим утилитам для очистки системы, программа отличается многообразием функций, нехарактерным для бесплатного приложения. Интерфейс CCleaner наглядно структурирован для осуществления основной задачи: очистки следов пользователя в Internet Explorer, Firefox и Opera, а также других разделах системы и сторонних приложениях.

CCleaner, в отличие от конкурентов, поддерживает не столь широкий ряд программ, но стирает историю популярных приложений, таких как Spybot - Search & Destroy и WinZip. В CCleaner теперь есть функция безопасного удаления файлов и инструмент для очистки реестра с предварительной проверкой. Перед удалением программа при необходимости сделает резервную копию реестра на случай потери важного компонента. В дистрибутив входит стандартная утилита для деинсталляции приложений.

Поскольку CCleaner абсолютно бесплатно обеспечивает полный набор инструментов для очистки системы наряду с дополнительными функциями, он вполне подойдет любому пользователю.

p.s. Модификация программы файлом настроек “winapp2.ini” - добавляет поддержку для более 500 новых программ в обработку инструментами CCleaner. Просто скопируйте файл “winapp2.ini” в директорию с установленной программой CCleaner. p.p.s Недавно появилась версия продукта и для Mac OS. Там CCleaner придется конкурировать с CleanMyMac от MacPaw. MacPaw в свою очередь пришли на Windows буквально в мае 2011 года. Но об этом позднее. Пока мало информации о CleanMyPC я собрал. Текущая версия на сегодня CleanMyPC 1.5.6.0.
