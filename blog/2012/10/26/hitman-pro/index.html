HitmanPro 3.6.2 Build 173 Final

Oct 26th, 2012

25 октября 2012 - HitmanPro 3.6.2 Build 173 Final.

<code>
<a href="http://www.surfright.nl/en/hitmanpro/whatsnew" target="_blank">HitmanPro Release History</a>:
Build 173 (2012-10-25)
• Исправлен режим "Force Breach" (не работал в билде 171).
• Улучшено обнаружение для семейства вредоносов "Ransom".
• Различные небольшие изменения.
• Исправлена ошибка, из-за которой иногда выключение сканера командной строки "зависало".
• Исправлена ошибка, при которой команда "/pup" не работала в сканере командной строки.
• Исправлена совместимость с Ashampoo firewall (32-bit).
• Улучшено обнаружение Bootkit'ов.
• Улучшена работа планировщика.
• Разные улучшения для сканера командной строки.
</code>

Если вы думаете, что ваш компьютер чист от вирусов, загрузите эту утилиту и проверьте. Хорошо если ваши предположения оправдались, но как минимум следящие куки будут найдены. Использует утилита “облачные технологии”, поэтому для проверки компьютера потребуется интернет соединение.

Краткая справка Сканер для выявления различных вредоносных программ. Особенность HitmanPro в том, что он использует при проверке базы данных других приложений для обнаружения и удаления шпионских модулей (Ikarus, G-Data, Emsisoft, Dr.Web, Prevx). Работает быстро и качественно.

Режим “Force Breach”, включается при нажатом левом Ctrl:

Загрузить программу можно с сайта производителя: • HitmanPro 3.6.2 Build 173 x86 • HitmanPro 3.6.2 Build 173 x64
