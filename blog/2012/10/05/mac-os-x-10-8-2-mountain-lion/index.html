Mac OS X 10.8.2 Mountain Lion [Update]

Oct 5th, 2012

04 октября 2012 года - Apple выпустила дополнительное обновление для операционных систем OS X Lion и OS X Mountain Lion. Минорный апдейт направлен на исправление ряда ошибок, которые возникли у некоторых пользователей после установки основного обновления. Сборка OS X 10.8.2. 19 сентября 2012 - релиз Mac OS X 10.8.2 Mountain Lion. Вы можете загрузить обновление в App Store и там же найти новый дистрибутив.

    1. 2012 Дополнительное обновление OS X 10.8.2: ● Исправление ошибки, которая могла приводить к неправильному отображению некоторых японских симполов в программе Mail. ● Возможность предоставлять Safari доступ к безопасным веб-сайтам при включенной функции родительского контроля. ● Исправление ошибки, которая могла препятствовать запуску система с более, чем 64 Гб оперативной памяти.
    1. 2012 Обновление 10.8.2 рекомендуется для установки всем пользователям OS X Mountain Lion и включает в себя исправления ошибок общего характера, направленные на повышение уровня стабильности и совместимости с аппаратным обеспечением Mac, а также добавляет ряд новых функций.

Особенности OS X Mountain Lion 10.8.2: Facebook ● Единая идентификация в Facebook. ● Возможность выбора Facebook для экспорта ссылок и фотографий. ● Контактная информация друзей Facebook и изображения профилей в программе Контакты. ● Уведомления Facebook в Центре уведомлений.

Game Center ● Экспорт счета игр в Facebook, Twitter, Mail или в Сообщения. ● Друзья Facebook включается в рекомендации друзей Game Canter. ● Кнопка Facebook «Нравится» для игр. ● Бросайте своим друзьям вызов побить ваши игровые рекорды или достижения.

Другие новые функции ● Поддержка Power Nap на MacBook Air (выпуска конца 2010 года). ● Сообщения iMessage, отправленные на ваш номер телефона, теперь появляются в программе Сообщения на вашем Mac. ● Вы можете добавлять новые карты в Safari и Mail на своем Mac в программу Passbook на своих iPhone или iPod touch, работающими под управлением iOS 6. ● Новые общие списки напоминаний. ● FaceTime теперь получает вызовы, отправленные на ваш номер телефона. ● Новые параметры сортировки позволяют сортировать заметки по названию, дате изменения и дате создания. ● Диктовка теперь поддерживает мандаринские и кантонское наречия китайского языка, испанский, корейский, канадский, английский, канадский французский и итальянские языки. ● Программа Словарь теперь включая в себя толковый словарь французского языка. ● Фотографии профилей Sina Weibo теперь могут быть добавлены в программу Контакты. необходима iOS 6

Важные исправления Обновление OS X Mountain Lion v10.8.2 включает в себя важные для операционной системы исправления, которые также улучшают стабильность, совместимость и безопасность вашего Mac’а. Вот часть из исправлений: ● добавлена опция для отмены изменений в исходном документе при выборе пункта меню «Save As»; ● при запуске почты теперь будут автоматически открываться неотправленные черновики; ● интеграция с Twitter научилась получать уведомления для упоминаний и ответов от любых пользователей; ● при отправки твитов из Центра уведомлений укорачиваются URL-адреса; ● в режиме AirPlay Mirroring уведомления отключаются автоматически; ● добавлена поддержка SSL для поиска Google через Safari; ● решена проблема, из-за которой чекбокс «Включить автообнаружение» (Enable Autodiscover) мог всегда оставаться выбранным; ● включен доступ к Mac App Store при активации Родительского контроля; ● добавлена поддержка адресов электронной почты @icloud.com; ● решена проблема с видео при подключении ноутбуков Mac к некоторым проекторам, оснащенным VGA-портом; ● устранена проблема, которая мешала блокировке учетных записей Active Directory; ● добавлены различные усовершенствования SMB; ● решены проблемы с автоматическим входом NIS-пользователей; ● устранены ошибки, при возникновении которых Keychain мог быть недоступен; ● добавлена возможность предварительной аутентификации защищенных систем FileVault; ● решена проблема, из-за которой Xsan автоматически не запускался после перехода со Snow Leopard.
