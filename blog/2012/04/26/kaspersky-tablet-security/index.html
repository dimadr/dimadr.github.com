Kaspersky Tablet Security 9.13.133

Apr 26th, 2012

26 апреля 2012 года Лаборатория Касперского объявляет о релизе Kaspersky Tablet Security версии 9.13.133 (для Android OS).

Улучшения в релизе Сritical Fix 1 (версия 9.13.133) для Kaspersky Tablet Security (Android): • Внедрена система push-уведомлений Google для отправки команд с веб-портала. Доставка команд на устройства c Tablet Security версии 9.13.133 происходит без задержки. • Исправлены ошибки с работой 3G на некоторых устройствах после установки Kaspersky Tablet Security.

Краткая справка Kaspersky Tablet Security Программа Kaspersky Tablet Security предназначена для обеспечения комплексной защиты планшетов. Пользователю предоставляются следующие возможности: • Постоянная проверка файловой системы на наличие подозрительной активности; • Проверка новых программ сразу после их установки; • Новинка! Дополнительная проверка новых программ до их первого запуска при помощи облачного сервиса Kaspersky Security Network; • Проверка файловой системы устройства по требованию пользователя или по расписанию на наличие вирусов и других вредоносных программ; • Лечение зараженных файлов, помещение на Карантин или их удаление, если лечение невозможно; • Обновление антивирусных баз программы по требованию пользователя или по расписанию (необходимо интернет-соединение); • Новинка! Возможность дистанционного блокирования устройства через Веб-портал, если устройство потеряно или украдено (см. скриншот); • Новинка! Возможность дистанционного удаления информации пользователя с устройства через Веб-портал, если устройство потеряно или украдено; • Новинка! Возможность дистанционного определения координат устройства на карте через Веб-портал, если устройство потеряно или украдено; • Новинка! Возможность дистанционного получения фотографий с фронтальной камеры устройства через Веб-портал, если устройство потеряно или украдено; • Новинка! Блокирование вредоносных и фишинговых веб-сайтов.

Системные требования: • Операционная система Android 2.2 - 4.х. • Разрешение экрана от 480x800 пикселей.

Установка программы производится только в основную память устройства. Тестирование Kaspersky Tablet Security проводилось на следующих устройствах: • Acer Iconia A100; • Acer Iconia A500; • Acer Iconia Tab 501; • Google Nexus S; • HTC Desire; • HTC Incredible S; • HTC Nexus one; • HTC Sensation; • HTC Vision; • LG Optimus P698 (DUAL SIM); • Motorola Xoom; • Samsung Galaxy 10.1; • Samsung Galaxy Nexus; • Samsung Galaxy S; • Samsung Galaxy S2; • Samsung Galaxy Tab; • Sony Ericsson Xperia Arc; • Sony Ericsson Xperia X10.

Основные известные ошибки: • Возможная задержка при синхронизации мобильного устройства с сервером (до 1 часа) • Кнопка Справка перекрывает элементы интерфейса программы при прокрутке.
