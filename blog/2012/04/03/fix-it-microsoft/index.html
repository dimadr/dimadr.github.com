Центр решений Microsoft Fix It

Apr 3rd, 2012

Компания Microsoft сделала доступным инструмент помощи и технической поддержки - центр решений Microsoft Fix it: устранение неполадок работы программного обеспечения. В задачи данного ресурса входит устранение проблем эксплуатации, инсталляции и обслуживания. Пользователи продуктов Майкрософт могут, выбрав соответствующий продукт, получить рекомендации по устранению проблем.
