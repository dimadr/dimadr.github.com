ESET Smart Security & NOD32 Antivirus 4.2.76.1 Business Edition Final

Jan 15th, 2012

15 января 2012 года выпущена новая версия антивирусов из линейки 4. Бизнес продукты требуют большего внимания в тестировании и ответственности производителя, поэтому линейка бизнес продуктов 5 подкрепляется проверенной надежностью четвертой версии. Антивирусные решения компании ESET популярны в России. Вот лишь несколько плюсов из достоинств этих продуктов:

а) небольшой размер дистрибутива; б) низкий процент загрузки процессора и ОЗУ; в) высокая эффективность относительно конкурентов; г) удобный интерфейс; д) расширенные настройки пользователя; е) небольшой размер обновлений сигнатур и т.д.

Лог изменений: NOD32 Antivirus & ESET Smart Security 4.2.76.1 • Fix: Occasional ekrn crash after installation • Fix: Occasional crashes of Windows Vista/Windows 7 during hibernation with firewall in interactive mode • Fix: Occasional issues with user profile readability • Fix: Certain websites opening blank in Internet Explorer • Fix: ThreatSense.Net settings not applied when product installed with an attached configuration file • Fix: Installer not detecting Microsoft Security Essentials present on the system • Fix: Compatibility issue with Steganos Safe software • Fix: Compatibility issue with Parallels Virtuozzo Containers • Fix: Issue with blocking of VPN use on Windows XP • Fix: Issue with establishing HTTP/POP3 connection • Fix: Issue with excessive creation of temp files by Mozilla Thunderbird • Fix: Detection of infected messages in Mozilla Thunderbird despite disabled e-mail client protection • Fix: Several localization issues • Added: Scanning of messages in Microsoft Outlook 2010 • Added: Support for Microsoft Windows Live Mail 2011

• Исправлено: случаи падения ОС после установки продукта ESET. • Исправлено: случаи падения ОС Windows Vista / Windows 7 в режиме “гибернации” с файерволом в интерактивном режиме. • Исправлено: ошибки чтения профиля пользователя. • Исправлено: случаи открытия “пустой страницы” в Internet Explorer. • Исправлено: ошибка в применении имеющихся настроек в ThreatSense.Net после установки продукта ESET. • Исправлено: инсталлятор ESET не определял установленный в системе Microsoft Security Essentials. • Исправлено: проблемы совместимости с программным обеспечением Steganos Safe. • Исправлено: проблема совместимости с Parallels Virtuozzo Containers. • Исправлено: ошибка блокирования VPN на Windows XP. • Исправлено: проблема с установлением соединения HTTP/POP3. • Исправлено: ошибка создания временных файлов Mozilla Thunderbird. • Исправлено: ошибка детектирования инфицированных сообщений в Mozilla Thunderbird при выключенном клиенте электронной почты. • Исправлено: ошибки локализации продукта. • Добавлено: сканирование сообщений в Microsoft Outlook 2010. • Добавлено: поддержка для Microsoft Windows Live Mail 2011.
