ESET Smart Security & NOD32 Antivirus 5.2.9.12 Final

May 16th, 2012

16 мая 2012 года - обновлена линейка продуктов ESET версии 5.2.9.1 English & 5.2.9.12 Russian Final.

Исправление в версии 5.2.9.1:

<code>
Общие изменения для Антивируса ESET NOD32 и ESET NOD32 Smart Security 
• Улучшено: Добавлена статистика проверенных файлов в главном окне антивируса.
• Улучшено: Добавлено больше информации о репутации в ESET Live Grid запущенных процессов.
• Поддержка: Возможность активировать продукт, используя прокси-сервер.
• Поддержка: Возможность использовать обновление ключей продукта.
• Добавлено: Настройки Блокирование съемных носителей и Антируткит (Зашита Anti-Stealth) представлены в окне главного интерфейса в меню Настройка.
• Исправлено: Перерисована тень "Быстрого меню" во время открывания.
• Исправлено: Графический интерфейс открывается в большом окне.
• Исправлено: Не релевантные всплывающие предупреждения показываются, когда не требуется никаких действий.
• Исправлено: Проблема с остановкой служб ESET в процессе установки.
• Исправлено: Ошибка запуска с HP Protect Tools, падение графического интерфейса ESET.
• Исправлено: Функция Блокирование съемных носителей не блокировала устройства для отдельных учетных записей Windows.
• Исправлено: Файлы продолжаются проверяться в режиме реального времени после отключения данной проверки.
• Исправлено: Планировка задачи обновления не может быть запущена из пользовательского интерфейса SysRescue.
• Исправлено: Несколько сбоев пользовательского интерфейса ESET при определенных обстоятельствах.
• Исправлено: Дополнительные настройки сканирования не работают Сканирование ПК.
• Исправлено: Активированный продукт сообщает, что он не активирован.
• Исправлено: Проверка протоколов IMAPS, POP3S и портов для выбранных приложений работает некорректно.
• Исправлено: Окно предупреждения об отключении некоторых компонентов защиты не отображается, если эти компоненты уже были отключены.
• Исправлено: Непоследовательность в статусе обновление базы данных вирусных сигнатур в пользовательский интерфейсе.
• Исправлено: Любой пользователь Windows может принудительно прервать запланированное задание.
• Исправлено: SSL-фильтрация в Chrome не работала в некоторых случаях. 

Изменения ESET NOD32 Smart Security 
• Улучшено: Добавлена информация в настройках Родительского контроля о необходимости включения SSL-фильтрации при фильтрации  Родительским контролем веб-сайтов с HTTPS.
• Улучшено: Небольшие изменения в настройках и управлении Родительского контроля.
• Улучшено: Администратор Родительского контроля может выбрать роль учетной записи Windows, используя меню настройки возраста.
• Улучшено: Информация о том, что Родительский контроль не фильтрует добавлено в главное окно.
• Добавлено: Для исключений для веб-страниц могут быть установлены приоритеты.
• Исправлено: Диалоговое окно профиля сети может показываться повторно.
• Исправлено: Проблемы с обнаружением в Windows Live Mail 2011.
• Исправлено: Некоторые страницы с определенной категорией оцениваются как "Без категории" Родительским контролем ESET Smart Security .
• Исправлено: Проблемы с импортом глобального списков Анти-спама из ESET Smart Security v4 в ESET Smart Security v5.
• Исправлено: Неправильные подсказки в диалоговых окнах Анти-спама.
• Исправлено: Родительский контроль показывается активным в пользовательском интерфейсе ESET, не смотря на отключение фильтрации протоколов.
• Исправлено: Проблемы с управлением исключениями фильтрации веб-страниц в Родительском контроле.</code>

Что нового в линейке 5: Основные преимущества: • Оптимизированная процедура запуска. Больше не нужно ждать загрузки базы данных новых сигнатур – ESET Smart Security 5 не позволит Вам начать работать c Вашим компьютером сразу после начальной загрузки, не ставя под угрозу Вашу безопасность. • Онлайновая безопасность. Учитывая всё увеличивающейся риск работы в Интернет, новая версия ESET Smart Security обнаруживает все типы переносимых Интернетом угроз. Эти включает фишинг в социальных сетях и социальных разработках, которые используют пользователи ежедневно. • Компактность системы. Сборка ESET NOD32 Antivirus была оптимизирована в течение многих лет в точном соответствии системным окружением. ESET сделали умное решение, которое мало нагружает систему, обеспечивая хорошую производительность.

Новые возможности • Родительский контроль. Родительский контроль позволяет Вам блокировать сайты, которые могут содержать потенциально опасный контент. Кроме того, родители могут запретить доступ к 20 предопределенным категориям веб-сайта. • Облачное сканирование. Принцип работы облачного сканирования нам уже знаком по другим продуктам. В ESET лаборатории теперь готовы в режиме реального времени получать и сканировать подозрительное и вирусное ПО. • Улучшена защита от спама. Наш антиспам-модуль интегрированный в ESET Smart Security претерпела значительный процесс по оптимизации работы и имеет еще более высокий уровень точности обнаружения. • Улучшение контроля медиа-устройств. Некоторые угрозы попадают в систему через съемные носители. ESET Smart Security 5 автоматически предложит вам просканировать содержание всех Flash USB дисков, компакт-дисков (CD и DVD). Сканирование будет также учитывать ряда параметров, такие как идентификатор устройства, тип носителя, а также его серийный номер. • Интеллектуальный Firewall. ESET Smart Security 5 помогает предотвратить несанкционированный доступ к компьютеру удаленно и контролирует сетевой трафик.

Новые возможности для гиков: • Режим игрока. В полноэкранном режиме ESET Smart Security автоматически переключается на тихий режим. Этот режим сохраняет системные ресурсы. Таким образом, Вы сможете играть в компьютерные игры или сконцентрироваться на задачах важной работы, не отвлекаясь на всплывающие сообщения. • Усовершенствованная функциональность HIPS. Это для настоящих Гиков – теперь Вы можете настроить поведение системы более “тонко”: сможете определить правила для системного реестра, активных процессов и программ, и настроить свои условия безопасности. • Усовершенствованное меню настроек. Перепроектированное усовершенствованное меню настроек - для более интуитивной и более легкой навигации. • Оптимизированное управление лицензией. Выберите тип активации, которая подходит Вам лучше всего. Все методы активации были разработаны, чтобы потребовать наименьшего вмешательства пользователя, насколько это было возможно. • Новый графический интерфейс. Графическая изменен с целью облегчить использование продукта. Наиболее часто используемые опции доступны со всех экранов. Вы сможете убедиться в этом, например на функции “Tools”, которая теперь еще более интуитивна и понятна.
