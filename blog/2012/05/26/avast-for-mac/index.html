Avast! Free Antivirus 7.0 Build 36987

May 26th, 2012

25 мая 2012 года обновился бесплатный антивирус для Mac OS X - avast! Free Antivirus 7.0 build 36987.

Загрузить avast! Free Antivirus 7.0 Build 36987

avast! Free Antivirus for Mac предлагает три уровня защиты и модуль сканированния по требованию (on-demand). Web фильтр и фильтр Mail вместе проверяют входящие данные из Интернет, в то время как модуль “Файловой защиты” сканирует все программы и выборочно файлы на вашем диске.

Возможности: • On-demand сканирование: этот модуль может сканировать всю файловую систему, сетевые диски и внешние примонтированнные устройства. Вы можете использовать выборочное сканирование. Отчет о сканировании автоматически будет создан по окончании и заархивирован, доступ к отчёту вы сможете получить из меню антивируса в любое время. • Предупреждения на экране: при обнаружении вредоносных объектов, сообщение предупреждающее вас об опасности появиться на экране. • Модуль файловой защиты (File Shield): сканирует данные запущенных приложений и все файлы во время их модификации. • Модуль защиты интернет (Web Shield): контролирует и фильтрует весь входящий HTTP траффик с интернет сайтов. Соединение вашего веб-броузера перенаправляется на “Модуль защиты интернет (Web Shield)”. Модуль защиты интернет (Web Shield) соединяется с запрошенным вами веб-сервером и одновременно проверяет данные загружаемые на ваш компьютер на присутствие вирусов и троянов. Только проверенные антивирусом данные будут переданы браузеру, до того как они сохранятся и повредят ваш Mac. • Модуль защиты почты (Mail Shield): контролирует и фильтрует весь входящий с email серверов POP3 и IMAP траффик. It does not prevent saving of infected files (doing so could lead to irreversible data loss), it just detects them. The File Shield prevents any infected attachments from causing harm to your system. • Автоматичекое обновление: Механизм сканирования автоматически обновляется регулярными обновлениями баз данных VPS.

Installation & Uninstallation: • To use the Mail Shield, you need to disable encrypted POP3/IMAP connections in your email client’s settings so the traffic can be scanned. By default, the Mail Shield encrypts all outgoing traffic itself if the mail server supports it (encrypted connections can be forced for selected mail accounts in the Mail Shield settings). • Uninstallation must be carried out from the application’s menu bar - item “Uninstall avast!”
