Microsoft Security Bulletin для Windows XP SP3 от 8 мая 2012 г.

May 11th, 2012

11 мая 2012 года - версия наборов для Windows x64. 09 мая 2012 года - обновлены наборы для Windows XP SP3 и для Windows 7 x86 до версии 12.5.9. Microsoft Security Bulletin Advance Notification от 8 мая 2012 г. Предварительно - обновлений для Windows XP SP3 будет пять (одно из них MRT). Предварительно - обновлений для Windows 7 SP1 x86/x64 будет пять (одно из них MRT).

Обновление безопасности для Microsoft Silverlight 4.1.10329.0 (KB2690729) Обновление безопасности для Microsoft Silverlight 5.1.10411.0 (KB2636927) Обновление для системы безопасности Windows XP (KB2659262) Обновление системы безопасности для ActiveX Killbits для ОС Windows XP (KB2695962) Обновление для системы безопасности Windows XP (KB2676562) Обновление для системы безопасности Windows XP (KB2686509) Средство удаления вредоносных программ для ОС Microsoft® Windows® (KB890830)
