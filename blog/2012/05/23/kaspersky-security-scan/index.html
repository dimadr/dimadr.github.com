Kaspersky Security Scan 2.0

May 23rd, 2012

Kaspersky Security Scan 2.0 — это бесплатная программа для диагностики вашего компьютера на наличие установленного антивирусного программного обеспечения.

Если вы не уверены в том, что ваш компьютер защищен, вы можете запустить проверку в Kaspersky Security Scan 2.0.

Программа распознает вредоносные объекты на основании информации, находящейся в антивирусных базах, а также использует эвристический анализатор для обнаружения еще не известных угроз и сервис Kaspersky Security Network, который обеспечивает высокую степень реакции программы на новые виды угроз.

В результате проверки вашего компьютера при помощи Kaspersky Security Scan 2.0 будет сформирован отчет со следующей информацией о вашем компьютере: • Установлено ли на вашем компьютере антивирусное программное обеспечение. • Установлен ли на вашем компьютере Сетевой экран (Firewall, брандмауэр). • Включен ли на вашем компьютере Сетевой экран (Firewall, брандмауэр). • Обнаружены ли на вашем компьютере вредоносные объекты. • Обнаружены ли уязвимости в операционной системе и установленных программах. • Для обеспечения безопасности вашей работы в локальных сетях и интернете предназначен Сетевой экран (брандмауэр, firewall). Сетевой экран представляет собой программный комплекс, который проверяет данные, передаваемые через Интернет или локальную сеть, и, в зависимости от параметров • • Сетевого экрана, блокирует или разрешает эту сетевую активность.

В Kaspersky Security Scan 2.0 вы можете выполнить следующие действия: • запустить/приостановить проверку; • сформировать расписание запуска проверки; • посмотреть результаты проверки в отчете.

Kaspersky Security Scan 2.0 предназначена только для диагностики вашего компьютера. Она не лечит зараженные файлы и не удаляет вредоносные программы на вашем компьютере.
