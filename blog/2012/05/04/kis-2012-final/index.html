Kaspersky Internet Security & Kaspersky Antivirus 2012 12.0.0.374 Final [Update]

May 4th, 2012

04 мая 2012 года Лаборатория Касперского выпустила патч I для Kaspersky Internet Security 2012. 21 февраля 2012 года - ЛК выпустила новый патч “H” для продуктов линеки 2012 (подробности под катом). 09 ноября 2011 года - ЛК выпустила новый патч “G” для продуктов линеки 2012 (подробности под катом). 06 октября 2011 года - ЛК выпустила новый патч “F” для продуктов линеки 2012 (подробности под катом).

Патч I автоматически устанавливается с обновлениями баз Kaspersky Internet Security 2012 версии 12.0.0.374. Патч “I” устраняет следующие проблемы: • Добавлена поддержка Firefox 12. • Внутреннии исправления (доработки). • Падения при ‘исследовании’ приложений. • Проблемы с плагином VK для IE (при закрытии IE или вкладок в IE). • Проблемы с плагином WTB для Google Chrome (запуск и работа под несколькими аккаунтами, одновременный запуск разных версий).

Патч “H” устраняет следующие проблемы: • Некоторые проблемы интерфейса Kaspersky Internet Security 2012. • При добавлении кода активации в качестве резервного происходит его активация при действующей коммерческой лицензии. • Ошибка активации программы, связанная с добавлением кода активации по подписке при действующей коммерческой лицензии. • Ошибка автоматического обновления статуса подписки. • Проблему отображения плагинов Анти-Баннера, Виртуальной клавиатуры и Модуля проверки ссылок в веб-браузере Mozilla Firefox 9, 10, 11. • Проблему обновления баз, сопровождающуюся ошибками “Базы: повреждены” и/или “Лицензия: недействительна”. • Проблему возникновения ошибки после завершения работы веб-браузера Google Chrome при запущенном Kaspersky Internet Security 2012. • Проблему автоматической установки сертификата Лаборатории Касперского для веб-браузера Mozilla Firefox 9, 10. Пока патч доступен только через автообновление.

Патч “G” устраняет следующие проблемы: • Некоторые проблемы интерфейса Kaspersky Internet Security версии 12.0.0.374. • Проблему обновления баз программы. • Ошибка отображения срока действия лицензии программы после добавления резервного кода активации. • Ошибка отображения количества оставшихся дней и даты окончания действия лицензии. • Ошибку отображения ранее обнаруженных и устраненных угроз в списке Проблемы безопасности. • Проблему отображения плагинов Анти-Баннера, Виртуальной клавиатуры и Модуля проверки ссылок в веб-браузере Mozilla Firefox 8. • Проблему возникновения ошибки “Файл ключа поврежден” при попытке активации программы. • Проблему “зависания” MS Office Outlook 2007 при проверке вложенных в почтовые сообщения архивов больших размеров (более 30 Мб) компонентом Анти-Спам.

Патч “F” устраняет следующие проблемы: • Некоторые проблемы интерфейса Kaspersky Internet Security версии 12.0.0.374. • Проблему потребления большого объема памяти во время обновления баз. • Ошибка отображения срока действия лицензии программы после добавления резервного кода активации. • Проблему освобождения памяти после выполнения полной проверки компьютера (свыше 500 000 объектов). • Проблему отображения плагинов Анти-Баннера, Виртуальной клавиатуры и Модуля проверки ссылок в веб-браузере Mozilla Firefox 7.

Более 80 дней ожидания и вот Kaspersky Internet Security 2012 “говорит по-русски”. Kaspersky Internet Security русская версия & Kaspersky Antivirus 2012 12.0.0.374 Final русская версия. Начатое первого марта этого бета - тестирование новой линейки продуктов антивирусной защиты 2012 закончено. Сегодня финальная версия Kaspersky Internet Security & Kaspersky Antivirus 2012 12.0.0.374 Final.

Как видите по ссылкам находится только английский вариант продуктов, традиционно, к сожалению, русский релиз в августе 2011 года, предположительно, но думаю ссылка на технический релиз появиться раньше. Дистрибутивы не содержат антивирусных сигнатур, отсюда малый размер дистрибутива.

Краткий список основных улучшений и нововведений Антивируса Касперского 2012:

* Полностью переработан интерфейс. Кардинально изменено оформление и структура (смотрите скриншоты). Также отмечаются следующие изменения: добавлен раздел с информацией об облачной защите, добавлена форма поиска в главное окно, оптимизирован просмотр новостей Лаборатории Касперского, включена поддержка сенсорных экранов, улучшен дизайн и функциональность гаджета;

* Улучшена процедура поиска несовместимых приложений, что позволяет избежать конфликтов между ними;

* Улучшена технология System Watcher. Теперь она выдает информацию не только о текущей сессии, но и собирает информацию о предыдущих сессиях;

* Добавлена опция проверки репутации приложения, рейтинга и статистики перед его запуском;

* Добавлена дополнительная информация о категории сайта в Kaspersky URL Advisor;

* Произведен ряд улучшения для KSN (Kaspersky Security Network);

* Улучшена система загрузки обновлений. Теперь обновления загружаются только для активных компонентов, а прочие приложения имеют приоритет на использование ресурсов системы;

* Улучшена эффективность защиты от руткитов;

* Усовершенствована технология самозащиты антивируса;

* Существенно усовершенствована производительность продукта и уменьшено его влияние на работу системы и других программ;

* Значительно усовершенствованы функции обнаружения фишинговых сайтов;

* Модуль Родительского контроля теперь имеет быстрый доступ к просмотру отчетов и настройкам;

* Включена поддержка новых веб-браузеров;

* Упрощены процедуры активации и покупки приложения.
