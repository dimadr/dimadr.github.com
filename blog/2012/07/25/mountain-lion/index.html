Mac OS X 10.8 Mountain Lion

Jul 25th, 2012

Сегодня Apple выпустила долгожданное обновление операционной системы для компьютеров Mac - OS X 10.8 Mountain Lion. Новый релиз доступен в Mac App Store за $19.99. В Mountain Lion появился ряд приложений, хорошо знакомых мобильным пользователям iOS. В частности, программа для работы с системой обмена сообщениями iMessage, приложения для создания заметок и напоминаний.

Выход Mountain Lion еще больше сближает платформу для компьютеров Apple с мобильной системой iOS, используемой на iPhone и iPad.

Новая операционная система также получила поддержку голосового ввода текста в любых приложениях. В общей сложности, в Mountain Lion вошло более 200 различных нововведений, включая улучшенную интеграцию с iCloud, Facebook и Twitter.
