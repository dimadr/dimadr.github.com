Panda Global Protection 2013 6.00.01 / Panda Internet Security 2013 18.00.01 / Panda Antivirus Pro 2013 12.00.01

Sep 4th, 2012

03 сентября 2012 обновлена линейка антивирусных продуктов компании Panda Security. Выпущены следующие версии: Panda Global Protection 2013 6.00.01, Panda Internet Security 2013 18.00.01, Panda Antivirus Pro 2013 12.00.01.

Новые антивирусы Panda Security 2013 по сравнению с предыдущей линейкой включают в себя ряд улучшений и новых функций: ● Новый интерфейс пользователя “look and feel”, который упрощает использование инструментов, содержащихся в продуктах 2013. ● Усовершенствованный антивирусный движок с еще более высоким уровнем обнаружения и возможностями лечения. ● Новый менеджер паролей: зашифрованное хранилище паролей, защищающее Ваши цифровые персональные данные и позволяющее Вам подключаться к веб-сервисам одним кликом. ● Новый инсталлятор, который значительно упрощает и ускоряет процесс установки продукта. ● Новая функция для автоматического выключения компьютера после проведения сканирования. ● Совместимость с Windows 8 Release Preview - бета-версией новой операционной системы от Microsoft. ● Новая система отправления запросов в Службу технической поддержки. Позволяет быстрее решать Ваши проблемы. ● Улучшенная система обнаружения уязвимостей в операционной системе. ● Повышенный уровень обнаружения вредоносных программ. ● Усовершенствования в различных модулях(TruPrevent, веб-прокси и файервол). ● Упрощенные и ускоренные процессы установки и активации продукта. ● Улучшенный интерфейс: добавлены новые кнопки, появилась возможность доступа к отчетам и статистике из окна сканирования. ● Улучшенные возможности лечения в Windows Vista, Windows 7 и Windows 8 (Consumer Preview). ● Улучшенное сканирование по запросу в результате использования всех ядер микропроцессора (мульти-поточное сканирование).

Действующая промо - акция для Panda Internet Security 2013.
