Microsoft Security Essentials 2.1.1116.0 Final

Feb 17th, 2012

16 февраля 2012 - компания Майкрософт опубликовала информацию о намерении обновить в своих антивирусных продуктах движок детектирования вирусов. Обновление планируется 20 февраля 2012 года. Напомню, что сейчас идет закрытое бета - тестирование новой версии Microsoft Security Essentials 4.0.1111.0 Beta. Beta версия использует тот же антивирусный движок что и общедоступный релиз. 28.06.2011 компания Майкрософт порадовала поклонников своего антивирусного продукта Microsoft Security Essentials выходом новой версии - 2.1.116.0 Final. С 30 марта ожидалась публикация новой версии с изменениями. И вот 28 июня - финальный релиз.

Загрузить Microsoft Security Essentials 2.1.116.0 можно на официальной странице продукта, выбрав свою версию операционной системы.

Новое в версии 2.0 Последняя версия содержит большие изменения. Наиболее важным, считаю, улучшения движка антивируса, а именно улучшение анализа эвристического сканирования и оптимизация производительности. Интересной новинкой является интеграция в систему и новая функция для брандмауэра - инспекция трафика. Эта функциональность встроена в Windows Vista и Windows 7 - механизм Windows Filtering Platform (функция недоступна в Windows XP). Microsoft Security Essentials 2.0 также интегрируется в веб-браузер, но только в Internet Explorer.

Краткая справка Microsoft Security Essentials пришел на смену Windows Live OneCare. Мы помним, что Windows Live OneCare занимал далеко не самые достойные позиции в рейтингах антивирусных решений, но надеемся, что разработчики сделали выводы и учли прежние ошибки. Как бы там ни было, а ближайший опыт пользователей и независимых рейтингов покажет результаты.

Microsoft Security Essentials обеспечивает защиту от вирусов, шпионских программ и прочего вредоносного программного обеспечения. Обновление антивирусной базы программы осуществляется через сервис Windows Update. Новинка имеет очень простой и доступный интерфейс, с которым справится даже неопытный пользователь. И, что очень важно, распространяется абсолютно бесплатно.

Необходимо отметить, что при установке, программа проверяет подлинность операционной системы и пользователям нелицензионной версии Windows не удастся ею воспользоваться.
