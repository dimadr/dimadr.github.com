Windows 8 Consumer Preview Build 8250 / Windows Server 8 Beta

Feb 29th, 2012

В середине сентября прошлого года публике впервые была представлена Windows 8 Developer Preview и вот сегодня компания Майкрософт представляет Windows 8 Consumer Preview - пользовательский билд для ознакомления с будущей операционной системой. Загрузить ознакомительную версию и прочитать подробнее новость вы можете на официальной странице. Страница для профессионалов - Windows Server 8 Beta.
