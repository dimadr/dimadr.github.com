Microsoft Security Essentials официально на русском v.1.0.1759.0

Dec 16th, 2009

Буквально одной строкой, спешу сообщить вам новость про Microsoft Security Essentials. Так как отзывов на мою статью было не мало, я понял, что вас заинтересовала тема об этом антивирусе. Так вот с сегодняшнего дня, многие интересовались, Microsoft Security Essentials теперь на русском для всех версий Windows операционных систем. Скачать можно с домашней страницы антивируса http://www.microsoft.com/security_essentials/?mkt=ru-ru. Помните, что работает данный антивирус только на лицензионной операционной системе.

p.s. 22 сентября 2010 года

Ранее бесплатный антивирус Microsoft Security Essentials (MSE) можно было использовать только дома для личных нужд. Лицензия не позволяла использовать этот антивирус на предприятиях и в компаниях.

Однако Microsoft делает шаг навстречу малому бизнесу и объявляет о смене правил лицензирования. Отныне, компании могут устанавливать до 10 копий антивируса на свои компьютеры. Для небольших фирм этого вполне достаточно.
