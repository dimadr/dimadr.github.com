F-Secure Mobile Security 6 месяцев бесплатно

Dec 24th, 2010

Компания F-Secure в содружестве с Facebook.com объявили акцию - зарегистрируйся и получи антивирус F-Secure Mobile Security на 6 месяцев бесплатно. Количество участников ограничено (5000 человек). Для участия в акции пройдите по этой ссылке (нужно быть авторизованным пользователем Facebook) и получите серийный номер на свой email.

Краткая справка F-Secure Mobile Security — комплексное решение для защиты данных, хранящихся на мобильных устройствах. В случае внедрения шпионских программ во время веб-сёрфинга и даже потери, кражи или выхода из строя вашего коммуникатора персональная конфиденциальная информация по-прежнему будет надёжно скрыта от посторонних глаз. F-Secure Mobile Security обеспечит владельцам смартфонов комфортное пребывание в сети.

Ключевые возможности F-Secure Mobile Security:

* Полноценная антивирусная защита для смартфонов с быстрым автообновлением баз. Антивирусная защита и сетевой экран в комплекте.

* Несколько уровней сканирования.

* Полная сохранность вашей информации.

* Защита от вирусов и вредоносных программ.

* Предотвращение распространения вирусов.

* Новая функция безопасной загрузки веб-контента, позволяющая вычислять потенциально опасные сайты. Нежелательные страницы автоматически блокируются.

  • Блокировка устройства при смене SIM-карты.

  • Встроенный GPS локатор F-Secure Mobile Security для определения местонахождения вашего мобильного устройства и данных его нового владельца (в случае утери).

  • Функционируя в фоновом режиме, F-Secure Mobile Security автоматически сканирует входящий и исходящий сетевой трафик, не требуя участия пользователя.

  • Критические обновления через SMS в случае недоступности интернет-соединения.

* Экономия места на диске, расхода энергии и памяти.

С помощью решения F-Secure Mobile Security for Business с антивирусом и файрволом IT-администраторы могут централизованно отслеживать степень защищённости всех смартфонов компании через единый сервисный портал. Поддерживается возможность подключения дополнительных смартфонов по беспроводной сети. Администратор может выслать код для активации через SMS-сервис и управлять статусом подписки через веб-браузер с возможностью дистанционной блокировки.
