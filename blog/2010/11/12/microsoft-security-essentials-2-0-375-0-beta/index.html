Microsoft Security Essentials 2.0.522.0 Beta

Nov 12th, 2010

Сегодня опубликована бета следующей версии Microsoft Security Essentials. Microsoft Security Essentials был выпущен в сентябре 2009 года, получивший признание за небольшой размер, эффективность и абсолютную бесплатность.

Microsoft Security Essentials создан для содействия решению текущих потребностей в области безопасности компьютеров, работающих с подлинными Windows - помочь людям сохранить защиту от вирусов, шпионского ПО и других вредоносных программ. Новые возможности бета Microsoft Основы безопасности включают в себя: Windows Firewall integration - улучшена интеграция с Windows Firewall, при установке MSE спросит о то, хотите ли вы оставить Firewall включенным или захотите выключить его. Enhanced protection for web-based threats - улучшена защита от веб-угроз и атак, MSE интегрируется с Internet Explorer для того, чтобы повысить безопасность интернет серфинга. New protection engine - новый движок защиты от вредоносного ПО. Новый движок обладает лучшим распознаванием вредоносного ПО вместе с тем обеспечивает еще большую производительность. Network inspection system - в MSE теперь интегрирована система инспекции сетевого окружения, которая защищает от сетевых эксплойтов. Для пользователей Windows XP: Обратите внимание, что система инспекции сетевого окружения не будет включена в Windows XP. Это потому, что эта система инспекции сетевого окружения требует Windows Filtering Platform (WFP) для того, и WFP доступна только в Windows Vista и Windows 7. Однако, если вы работаете в Windows XP, вы можете использовать все другие возможности бета-версии.

Загрузить обновленную версию MSE Beta можно с сайта Connect по этому адресу.

p.s. (Май 2010) тестирование Microsoft Security Essentials 1.0.1963.0

В мае 2010 года группой антивирусных экспертов AV-Comparative.org было проведено очередное сравнительное тестирование антивирусных продуктов. Среди тестируемых продуктов оказался и Microsoft Security Essentials. Результаты высокие.

  19. Trend Micro AntiVirus plus AntiSpyware 2010
  20. Trustport2 Antivirus 2010 Большая часть соперников — платные решения, требующие ежегодных платежей для получение обновлений сигнатурных баз, а также обновлений ядра программы. Исключения — avast! Free Antivirus 5.0, возможно, AVG Anti-virus (он бывает и бесплатный). И, конечно же, сам Microsoft Security Essentials. Авторы тестирования хотели проверить степень развитости и успешности системы проактивного детектирования в современном антивирусном ПО. Проактивная защита — это специальная технология, позволяющая анализировать поведение и выявлять потенциально опасные программы. Основные функции — находить вирусы без запуска сканнирования, предотвращать заражение компьютера зловредом и другое. Тестирование проводилось с использованием свежих вирусных образцов, еще не доступных в сигнатурных базах. По результатам исследования Microsoft Security Essentials получил статус ADVANCED+, обезвредив 59% процентов вредоносных программ с помощью системы проактивной защиты. Security Essentials признал за вирус нормальные файлы всего 3 раза (очень выскоий результат для подобного рода программ). MSE разделил 3-е место с антивирусом Касперского.
